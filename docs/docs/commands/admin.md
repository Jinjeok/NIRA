# 관리/시스템

봇 관리 및 시스템 기능들

## `/메시지삭제`

**설명:** 채널 메시지 일괄 삭제

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| amount | integer | ✅ |  | 삭제할 메시지 수 |  |

**사용법:**
```
/메시지삭제 <amount>
```
