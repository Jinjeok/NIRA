# 재미/게임

주사위, 동전, 추첨 등 재미있는 기능들

## `/선택`

**설명:** 여러 항목 중 하나를 랜덤으로 선택하거나, 확률에 따라 선택해줍니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 항목 | string | ✅ |  | 선택할 항목들을 쉼표(,)로 구분하여 입력해주세요. (예: 밥, 국수, 빵) |  |
| 확률 | string |  |  | 각 항목의 확률을 쉼표(,)로 구분하여 입력해주세요. (예: 50,30,20). 항목 개수와 일치해야 합니다. |  |

**사용법:**
```
/선택 <항목> [확률]
```


---

## `/복권`

**설명:** 로또 등수를 랜덤으로 뽑거나 확률표를 보여줍니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 확률표보기 | boolean |  |  | 확률표만 보려면 true로 설정하세요. (기본값: false) |  |

**사용법:**
```
/복권 [확률표보기]
```

---

## `/홀짝`

**설명:** 무작위 숫자의 홀/짝을 맞춰보세요.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 선택 | string | ✅ |  | 홀과 짝 중에 하나를 선택합니다. | 홀(odd)<br/>짝(even) |

**사용법:**
```
/홀짝 <선택>
```

## `/랜덤`

**설명:** 지정된 숫자 사이의 랜덤한 값을 출력합니다

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 최소 | integer | ✅ |  | 최소값 |  |
| 최대 | integer | ✅ |  | 최대값 |  |

**사용법:**
```
/랜덤 <최소> <최대>
```

---

## `/확률`

**설명:** 입력된 확률이 몇번만에 성공하는지 확인합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 횟수 | integer |  |  | 고정 시도 횟수 (입력 시 해당 횟수만큼 시도, 없으면 첫 성공까지 시도) |  |
| 확률 | number | ✅ |  | % 확률 |  |

**사용법:**
```
/확률 <확률> [횟수]
```
