# 명령어 목록

NIRA가 제공하는 모든 슬래시 명령어입니다.

## 소셜/커뮤니티

투표, 아바타 등 소셜 기능들

- **`/아바타`** - 본인의 아바타를 출력합니다.
- **`/찬반`** - 찬성/반대 시뮬레이션
- **`/트위터`** - X 또는 Twitter 링크를 vxtwitter 링크로 변환합니다.

## 유틸리티

계산, 변환, 시간 등 일상적인 도구들

- **`/base64`** - Base64 코드/평문 을(를) 인코딩/디코딩 합니다.
- **`/계산`** - 수식을 계산합니다.
- **`/시계`** - 현재 시각 표시
- **`/컬러`** - 랜덤 또는 지정된 색을 출력합니다.
- **`/환율`** - 환율을 조회하고 환산합니다.
- **`/길이`** - 입력 문자열의 길이를 측정합니다.

## 재미/게임

주사위, 동전, 추첨 등 재미있는 기능들

- **`/선택`** - 여러 항목 중 하나를 랜덤으로 선택하거나, 확률에 따라 선택해줍니다.
- **`/코인`** - 빗썸 코인 시세 조회
- **`/복권`** - 로또 등수를 랜덤으로 뽑거나 확률표를 보여줍니다.
- **`/홀짝`** - 무작위 숫자의 홀/짝을 맞춰보세요.
- **`/오미쿠지`** - 운세를 테스트해볼 수 있습니다.
- **`/랜덤`** - 지정된 숫자 사이의 랜덤한 값을 출력합니다
- **`/확률`** - 입력된 확률이 몇번만에 성공하는지 확인합니다.

## 기타

기타 다양한 기능들

- **`/디데이`** - 오늘과의 일수 차이 계산
- **`/이모지변환`** - 영어를 이모지로 변환
- **`/제미나이`** - Gemini AI에게 질문합니다. (대화 내용은 24시간동안 유지됩니다. 페르소나별 대화 독립됨)
- **`/마트휴무일`** - 오늘의 마트 의무휴업일 정보를 확인합니다. (매월 2, 4번째 일요일 기준)
- **`/소비세`** - 엔화 금액의 소비세 및 원화 환산 가격을 계산
- **`/나이`** - 세는 나이 계산 (YYYY)

## 관리/시스템

봇 관리 및 시스템 기능들

- **`/메시지삭제`** - 채널 메시지 일괄 삭제
- **`/홈페이지`** - NIRA의 홈페이지 링크를 출력합니다.
- **`/초대하기`** - NIRA 봇을 서버에 초대하는 링크를 생성합니다.
- **`/핑`** - NIRA 봇이 살아있는지 확인합니다.
- **`/강화`** - 강화 시뮬레이션

## 정보 조회

주식, 환율, 핫딜 등 실시간 정보 조회

- **`/핫딜`** - [최신 핫딜을 여기에서 확인하세요](https://www.ppomppu.co.kr/zboard/zboard.php?id=ppomppu)
- **`/뉴스레터`** - 선택한 종류의 뉴스레터를 가져와 보여줍니다.
- **`/금액`** - 숫자를 한글 금액 표현으로 변환합니다.
- **`/주식`** - 지정한 종목의 현재 주식 정보를 검색합니다.

## 텍스트 처리

텍스트 변환, 처리 관련 기능들

- **`/문자열`** - 랜덤 영어 / 한국어 / 일본어 문자열을 생성합니다.
- **`/거꾸로`** - 문장을 거꾸로 출력
- **`/말`** - 메시지를 NIRA가 대신 보냅니다.
- **`/그긴거`** - 그 긴거 목록에서 특정 항목을 선택하여 보거나 전체 목록을 봅니다.
