# 정보 조회

주식, 환율, 핫딜 등 실시간 정보 조회

## `/핫딜`

**설명:** [최신 핫딜을 여기에서 확인하세요](https://www.ppomppu.co.kr/zboard/zboard.php?id=ppomppu)

**파라미터:** 없음

**사용법:**
```
/핫딜
```

---

## `/뉴스레터`

**설명:** 선택한 종류의 뉴스레터를 가져와 보여줍니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 분야 | string | ✅ |  | 보고 싶은 뉴스레터 분야를 선택하세요. |  |

**사용법:**
```
/뉴스레터 <분야>
```

---

## `/금액`

**설명:** 숫자를 한글 금액 표현으로 변환합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 액수 | string | ✅ |  | 숫자만 입력 (쉼표 없이) |  |

**사용법:**
```
/금액 <액수>
```

---

## `/주식`

**설명:** 지정한 종목의 현재 주식 정보를 검색합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 종목명 | string | ✅ |  | 검색할 주식의 이름 (예: 삼성전자) |  |
| 상세 | boolean |  |  | 상세 정보를 보려면 true로 설정하세요. (기본값: 간략 정보) |  |

**사용법:**
```
/주식 <종목명> [상세]
```
