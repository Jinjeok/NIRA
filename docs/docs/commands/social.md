# 소셜/커뮤니티

투표, 아바타 등 소셜 기능들

## `/아바타`

**설명:** 본인의 아바타를 출력합니다.

**파라미터:** 없음

**사용법:**
```
/아바타
```

---

## `/찬반`

**설명:** 찬성/반대 시뮬레이션

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| times | integer | ✅ |  | 횟수 |  |

**사용법:**
```
/찬반 <times>
```

---

## `/트위터`

**설명:** X 또는 Twitter 링크를 vxtwitter 링크로 변환합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| url | string | ✅ |  | 변환할 X 또는 Twitter 링크 |  |

**사용법:**
```
/트위터 <url>
```
