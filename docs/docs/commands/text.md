# 텍스트 처리

텍스트 변환, 처리 관련 기능들

## `/문자열`

**설명:** 랜덤 영어 / 한국어 / 일본어 문자열을 생성합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 언어 | string | ✅ |  | 언어 출력 선택 | 영어(en)<br/>한국어(ko)<br/>일본어(jp) |
| 길이 | integer |  |  | 출력할 문자열의 개수 (최대 2000) |  |

**사용법:**
```
/문자열 <언어> [길이]
```

---

## `/거꾸로`

**설명:** 문장을 거꾸로 출력

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| content | string | ✅ |  | 입력할 문장 |  |

**사용법:**
```
/거꾸로 <content>
```

## `/말`

**설명:** 메시지를 NIRA가 대신 보냅니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 메시지 | string | ✅ |  | 보낼 메시지 |  |

**사용법:**
```
/말 <메시지>
```

---

## `/그긴거`

**설명:** 그 긴거 목록에서 특정 항목을 선택하여 보거나 전체 목록을 봅니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 항목 | string | ✅ |  | 보고 싶은 항목을 선택하세요. |  |

**사용법:**
```
/그긴거 <항목>
```
