# 유틸리티

계산, 변환, 시간 등 일상적인 도구들

## `/base64`

**설명:** Base64 코드/평문 을(를) 인코딩/디코딩 합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 모드 | string | ✅ |  | 인코딩(e) 또는 디코딩(d) | 인코딩(e)<br/>디코딩(d) |
| 값 | string | ✅ |  | 처리할 문자열 |  |

**사용법:**
```
/base64 <모드> <값>
```

## `/계산`

**설명:** 수식을 계산합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 수식 | string | ✅ |  | 계산할 식 (예: 2 * (3 + 4) / 5^2 - sqrt(9)) |  |

**사용법:**
```
/계산 <수식>
```

## `/시계`

**설명:** 현재 시각 표시

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 시차 | integer |  |  | ±시간 (정수) | KST기준 PST = -16, UTC = -9 |  |

**사용법:**
```
/시계 [시차]
```

**설명:** 랜덤 또는 지정된 색을 출력합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| 16진수 | string |  |  | 16진수 색상 (예: ff0000) |  |
| 미리보기 | boolean |  |  | 이미지 미리보기 여부 |  |

**사용법:**
```
/컬러 [16진수] [미리보기]
```

## `/환율`

**설명:** 환율을 조회하고 환산합니다.

**파라미터:**

| 이름 | 타입 | 필수 | 기본값 | 설명 | 선택지 |
|---|---|---|---|---|---|
| pair | string | ✅ |  | 환산할 통화 쌍을 선택하세요. | KRW → JPY(kj)<br/>JPY → KRW(jk)<br/>KRW → USD(ku)<br/>USD → KRW(uk) |
| amount | number |  |  | 기준 통화 단위 (기본 단위 자동) |  |

**사용법:**
```
/환율 <pair> [amount]
```
